# Personal Codex Skills

This repository stores personal Codex skills and their design notes.

## Skills

- `ssh-linux-diagnose`: Diagnose Linux servers over SSH or manual command relay with interview-first triage, AI second review for every remote command, and human confirmation for state-changing commands or high-risk command text.

## Install

Copy a skill directory into your personal Codex skills folder:

```bash
mkdir -p ~/.agents/skills
cp -R skills/ssh-linux-diagnose ~/.agents/skills/
```

## Layout

- `skills/`: installable skill directories.
- `docs/plans/`: approved designs and implementation plans.
