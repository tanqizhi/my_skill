# External assessment

Use Nmap for bounded port/service discovery and Nuclei for selected vulnerability and exposure templates. When an in-scope target is a Web business, offer Chaitin Xray as an additional Web scanner.

Generate structured outputs for later parsing. Use conservative rates and explicit timeouts. Never enable brute force, denial of service, destructive fuzzing, cloud upload, target expansion, or unsafe PoCs. In report-directed mode, validate exact findings and disable broad Nmap discovery and Xray crawling.
